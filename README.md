# AI Financial Data Analyzer

This project analyzes financial trading datasets using SQL and generates AI-powered insights using LLM APIs.

## Features

- Load trading data into PostgreSQL
- Run SQL analytics queries
- Generate AI-powered summaries
- Detect trading patterns

## Tech Stack

- Python
- PostgreSQL
- Pandas
- OpenAI API
- SQL
